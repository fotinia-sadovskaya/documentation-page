Introduction to JavaScript

JavaScript is a versatile programming language used for web development.

It allows developers to add interactivity to websites.

JavaScript can be used for both frontend and backend development.

It is an essential part of modern web applications.

Browsers have built-in JavaScript engines to execute JS code.

console.log("Hello, JavaScript!");
Variables and Data Types

JavaScript has three main ways to declare variables: var, let, and const.

There are different data types in JavaScript:

  • String
  • Number
  • Boolean
  • Object
  • Array
  • Undefined
  • Null
let age = 25;
Functions in JavaScript

Functions allow code reuse and modularization.

A function can be defined using the function keyword or arrow syntax.

Functions can take parameters and return values.

function greet(name) { return "Hello, " + name; }

JavaScript also supports anonymous and callback functions.

Loops and Iterations

Loops help in executing repetitive tasks efficiently.

JavaScript supports several types of loops:

  • for loop
  • while loop
  • do-while loop
  • forEach method (for arrays)
for (let i = 0; i < 5; i++) { console.log(i); }
DOM Manipulation

The Document Object Model (DOM) allows interaction with HTML elements.

Common DOM methods:

  • document.getElementById()
  • document.querySelector()
  • document.createElement()
  • element.appendChild()
  • element.innerHTML
document.getElementById("title").textContent = "New Title";
